# 0.1.0 — 2026-06-04

## Added

- **`eurlex_lookup_celex`** — resolve CELEX, ELI, or OJ identifiers to canonical CELLAR work URIs
- **`eurlex_browse_subjects`** — search EuroVoc thesaurus; prerequisite for subject-filtered searches
- **`eurlex_search_documents`** — search EU legislation and preparatory acts with type, date, EuroVoc, and in-force filters
- **`eurlex_get_cases`** — search CJEU and General Court case law by case number, court, and case type
- **`eurlex_get_document`** — fetch structured metadata and full HTML/XML text via the EUR-Lex content API
- **`eurlex_get_relations`** — traverse the CELLAR CDM relationship graph (amendments, consolidations, citations, legal basis)
- **`eurlex_query_sparql`** — raw SPARQL SELECT escape hatch with server-side LIMIT injection/cap at 100
- **`eurlex://document/{celexNumber}`** resource — metadata snapshot for a CELLAR work
- **`eurlex://document/{celexNumber}/relations`** resource — relationship summary for a CELLAR work
- **`eurlex_comparative_analysis`** prompt — comparative EU ↔ US legal analysis template
- **`CellarSparqlService`** — POST client for the CELLAR Virtuoso SPARQL endpoint with LIMIT enforcement and Virtuoso error detection
- **`EurLexContentService`** — GET client for the EUR-Lex legal-content URL pattern with language fallback
