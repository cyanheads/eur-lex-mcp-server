# 0.9.4 — 2026-07-04

## Changed

- **`eurlex_browse_subjects` `language`** ([#46](https://github.com/cyanheads/eur-lex-mcp-server/issues/46)) — schema now accepts case-insensitive 2-3 letter codes (`/^[A-Za-z]{2,3}$/`, matching `eurlex_get_document`'s existing pattern); the handler's lowercase normalization before the SPARQL query is unchanged.
- **`eurlex_get_document` `offset`/`outline`/`select`** ([#48](https://github.com/cyanheads/eur-lex-mcp-server/issues/48)) — input descriptions now state that offsets and section location are format-specific; keep `format` constant when paging from an outline offset.
- **`eurlex_get_document` `is_superseded`** ([#49](https://github.com/cyanheads/eur-lex-mcp-server/issues/49)) — output description reworded to lead with "a newer consolidated version exists" and explicitly disclaim repeal/replacement; the base act remains the law (see `in_force`). No schema or behavior change.
